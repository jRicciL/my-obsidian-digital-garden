# Introduction to Power BI

## PowerBI Components
- **Desktop version:**
	- Data Analysis and report creation tool
	- includes the Power Query Editor
	- 100% Free
- **Power BI service:**
	- Cloud-based version
	- Light report editing
	- Share and distribute reports
